Projetos em Destaque

Distribuição de Alimentos

Este projeto realiza entregas semanais de cestas básicas para famílias em situação de vulnerabilidade social. Nosso objetivo é garantir alimentação digna e segurança alimentar para crianças e adultos.

Apoiar este projeto

Educação Infantil

Promovemos reforço escolar, atividades educativas e oficinas culturais. Esse projeto ajuda no desenvolvimento educacional e social das crianças.

Apoiar este projeto

Doação de Roupas

Distribuímos roupas, calçados e cobertores para famílias carentes, principalmente em épocas frias. Garantimos conforto e proteção para quem precisa.

Apoiar este projeto

Ações Comunitárias

Realizamos eventos sociais, mutirões solidários e atividades com voluntários. Essas ações fortalecem os laços comunitários e promovem inclusão social.
